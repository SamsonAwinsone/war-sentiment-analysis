⚠ Data Collection Challenges

Twitter / X API
Requires paid Elevated access ($100+/month). Free tier removed in 2023. Reddit used as an alternative social media source.
Facebook / Instagram
Meta Graph API requires business verification and app review. Public page scraping is blocked by login walls and anti-scraping measures.
RSS Rate Limits
RT News and some regional outlets throttle repeated requests. Polite crawl delays (0.4s) and error handling applied.
Paywalled Content
Full article text behind paywalls (NYT, FT). Only RSS summaries and headlines are accessible without subscriptions.
Multilingual Text
Arabic, Farsi, Hebrew content filtered to English-only for VADER compatibility. Helsinki-NLP translation could enable multilingual analysis.
Content Bias
RT News is government-affiliated; Al Jazeera is state-funded. Vocabulary and framing differ systematically from Western outlets.

◈ Topic Modeling LDA · gensim

📖 Methodology Note

LDA models each document as a mixture of topics and each topic as a mixture of words. We set α = auto (asymmetric Dirichlet prior) and run 10 passes for convergence. Preprocessing removes stopwords, URLs, numbers and tokens <3 characters. Trade-off: LDA assumes bag-of-words — word order and negation are lost.

◆ Sentiment Analysis VADER

VADER (Valence Aware Dictionary and sEntiment Reasoner) — unsupervised lexicon-based method. Compound scores range from −1 (most negative) to +1 (most positive).

📖 Why VADER?

VADER requires no labelled training data (unsupervised), handles social media idioms and punctuation-based emphasis, and provides a continuous compound score. Limitation: fixed English lexicon may mis-score domain-specific military/political vocabulary.

📖 Thematic Classification

Articles are classified into five thematic angles via keyword-density scoring. Each theme has a curated seed vocabulary (e.g. "missile, drone, strike" → Military Operations). A neural zero-shot classifier would improve precision on ambiguous articles.
