⚠ Data Collection Challenges

Twitter / X API
Requires paid Elevated access ($100+/month). Free tier removed in 2023. Solution: Reddit used as social media proxy.
Facebook / Instagram
Meta Graph API requires business verification & app review. Public page scraping blocked by login walls.
RSS Rate Limits
RT News and some regional outlets throttle requests. Exponential backoff and polite crawl delays applied.
Paywalled Content
Full article text behind paywalls (NYT, FT). RSS summaries used; NewsAPI snippets supplement where available.
Multilingual Text
Arabic, Farsi, Hebrew content filtered to English-only for VADER compatibility. Future: translate with Helsinki-NLP.
Content Bias
RT News government-affiliated; Al Jazeera state-funded. Framing and vocabulary differ systematically from Western outlets.

📖 Methodology Note

LDA models each document as a mixture of topics and each topic as a mixture of words. We set α = auto (asymmetric Dirichlet prior) and run 10 passes for convergence. Preprocessing removes stopwords, URLs, numbers and tokens <3 characters. Trade-off: LDA assumes bag-of-words so word order and negation are lost — topics may blend semantically distinct but co-occurring terms (e.g. "peace" and "war").

◆ Sentiment Analysis VADER

VADER (Valence Aware Dictionary and sEntiment Reasoner) — an unsupervised, lexicon-based method calibrated for short texts and social media. Compound scores range from −1 (most negative) to +1 (most positive).

📖 Why VADER?

VADER was chosen because: (1) it requires no labelled training data (unsupervised), (2) it handles social media idioms, capitalisation and punctuation-based emphasis, and (3) it provides a continuous compound score enabling granular analysis. Limitation: VADER uses a fixed English lexicon and may mis-score domain-specific military/political vocabulary. Comparison with TextBlob polarity scores would reduce lexicon bias.

📖 Thematic Classification

Articles are classified into five thematic angles using keyword-density heuristics — each theme has a curated seed vocabulary (e.g. "missile, drone, strike" → Military Operations). The category with the highest matching keyword count wins. This is a deterministic rule-based approach; a neural classifier (e.g. zero-shot via HuggingFace) would improve precision on ambiguous articles.
